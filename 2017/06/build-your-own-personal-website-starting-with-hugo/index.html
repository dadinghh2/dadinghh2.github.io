Build your own personal website starting with Hugo

June 1, 2017 in Web Development

Introduction of Hugo

Unlike other frameworks like Jekyll and Hexo, which are developed for personal blog building, Hugo is a general-purpose website framework, also a static site generator. Hugo is developed by Go language and has a very fast responsive speed. I chose Hugo because it allows the author to post not only blogs but also some other things like docs, portfolios, etc. Hope you will enjoy Hugo just like me!

Installing Hugo

Installation is very easy. Simply download the appropriate version for your platform from Hugo Releases. If you use open source system like Linux, you had better put the extracted excutable file in the /usr/local/bin location and add it to your PATH variable. On macOS, if you have Homebrew, just run brew update && brew install hugo. Here is a detail instruction for macOS installation made by the Hugo’s author. For windows users, please see Installation on Windows. You can test Hugo while you installed it with hugo help in a command prompt window. For a complete knowledge of Hugo, please visit the author’s Hugo docs website.
